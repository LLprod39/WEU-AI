Управление поведением ИИ при анализе и выполнении задач

dns

Подтверждение сервера

Когда ИИ определяет сервер для выполнения задачи, он сначала спросит подтверждение: «Буду выполнять на сервере X, верно?». Вы сможете подтвердить или выбрать другой сервер.

bolt

Авто-выполнение простых задач

Если задача определена как простая (complexity: simple) и сервер известен, ИИ может сразу приступить к выполнению без дополнительного подтверждения.

warning Используйте с осторожностью. Рекомендуется только для тестовых серверов.

help_outline

Уточняющие вопросы

Если ИИ не может определить все детали для выполнения задачи, он задаст уточняющие вопросы прежде чем приступить к работе.

computer

Сервер по умолчанию

Если в задаче не указан сервер, будет использоваться этот сервер. Оставьте пустым, чтобы ИИ всегда спрашивал какой сервер использовать.
