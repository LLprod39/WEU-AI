# Изображения для README и документации

## Презентация (WEU_AI_Agent_Platform_DevOps_Edition.pdf)

Чтобы картинки из презентации отображались в README на GitHub:

1. Откройте PDF в просмотрщике (браузер, Adobe, Foxit и т.п.).
2. Экспортируйте каждую страницу как PNG:
   - **Вариант А:** Скриншот каждого слайда (Win+Shift+S / PrtScn).
   - **Вариант Б:** В некоторых программах есть «Экспорт страниц как изображения».
3. Сохраните файлы в эту папку (`docs/images/`) с именами:
   - `presentation-01.png` — 1-й слайд  
   - `presentation-02.png` — 2-й слайд  
   - …  
   - `presentation-12.png` — 12-й слайд  

После добавления файлов они автоматически появятся в README на GitHub.
