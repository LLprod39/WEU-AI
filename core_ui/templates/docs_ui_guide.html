UI Guide

Визуальное руководство по интерфейсу и настройкам.

1. Интерфейс чата

В чате доступны: провайдер (Gemini / Grok), выбор модели и переключатель RAG.

  • Провайдер — при смене обновляется список моделей.
  • Модель — только модели выбранного провайдера, сохраняется в сессии.
  • RAG — добавляет контекст из базы знаний.

2. Настройки → Модели

Отдельные модели для чата, агента и RAG; провайдер по умолчанию.

3. Примеры

Быстрый чат: Чат → выбор провайдера и модели → сообщение.

Настройка под задачи: Настройки → Модели → задать модели для чата, агента, RAG → Сохранить.

4. Цвета и индикаторы

Gemini — primary; Grok — вторичный; RAG — emerald. Уведомления при смене провайдера/модели и при сохранении.
